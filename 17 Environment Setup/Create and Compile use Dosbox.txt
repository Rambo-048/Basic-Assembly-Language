CREATE AND COMPILE ASSEMBLY LANGUAGE USE DOSBOX

1. Create Source Code
2. Compile
   - TASM filename or filename.asm
3. linker
   - TLINK filename      (Create .EXE, Warning:No Stack)
   - TLINK /t filename   (Create .COM)
   - TLINK /T filename   (Fatal: Invalid Target /T)
4. Execute 
   - filename.com OR
   - filename.exe
